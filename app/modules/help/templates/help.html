Help and Information Popovers

Usage

  • Limit to 1-2 sentences.
  • Don’t use these to describe what a control does.
  • Use selectively so as not to overcrowd the screen with prompts.

Behavior

  • Prompt copy appears on click, not on hover.
  • A second click dismisses the prompt.

Symbol Selection

We offer two popover icons: a lowercase “i” (for “information”) and a question mark to denote help. In most cases, you will use a question mark. Here are criteria to help you recognize which icon to use for your popover.

Use the question mark when your popover:
  • Directly answers a question the user is likely to have.
  • Summarizes and/or links to more detailed help content available elsewhere.
Use the lowercase “i” when your popover:
  • Contains useful, but not essential, information about a control.
  • Offers additional explanatory information, but is not “help” per se.

Note that the “i” may not be meaningful in some countries. In these cases, we recommend that you use only the question mark icon in your localized product versions.

Tooltips

Usage

  • Limit to a single word or short phrase; sentence fragments are okay.
  • Use only when the purpose of an onscreen element isn’t self-evident.
  • If an element already has a descriptive label then don’t repeat that information in a tooltip.

Style

  • Don’t end tooltip copy with punctuation.
  • Use sentence capitalization except when a tooltip contains a proper noun (e.g., “Address Book”)
  • Use selectively so as not to overcrowd the screen with prompts.

Entrance/Exit ng- if/show/hide

Entrance/Exit class names*

Entrances and exits come in different flavors. These are some basic ones.

  • pb-fade: This is a catch all - this will work for any case, whether it be a pb-fade out or in.
  • pb-fadeIn: This will only work with ng-show or the entrance of an ng-if animation.
  • pb-fadeOut: This will only work with ng-hide or the exit of an ng-if animation.
* Prefix these class names with a . in your Sass file if using in an @extend.
